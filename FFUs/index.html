Flashing instructions

You will need:

Windows Device Recovery Tool WPinternals

1. Download Windows Device Recovery Tool and install it

2. Download WPinternals and extract it

3. Unlock the device using WPinternals, for a detailed guide, see here

4. Open WPinternals, switch the phone to Flash mode and make sure the phone is fully charged and close WPinternals after switching to Flash mode

5. Open a terminal window here: '\Program Files (x86)\Microsoft Care Suite\Windows Device Recovery Tool'

6. Run the following command: thor2 -mode uefiflash -ffufile 10.0.XXXXX.XXXX_CLIENTPRO_XXXXXX1_A64FRE_EN-US.ffu -erase_data

7. Run the following command: thor2 -mode rnd -reboot

1 Replace the filename with the one you downloaded.
